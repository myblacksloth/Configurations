# Workflow

```

```

## Implementazione

```
Use the coder subagent to implement the approved plan with minimal changes.
```

## Revisione

```
Use the reviewer subagent to audit the implementation for logic errors, weak error handling, and security issues.
```

## Test

```
Use the tester subagent to add or update pytest tests for the modified modules.
```

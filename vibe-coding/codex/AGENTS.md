# Codex Agent Workflow

Use this workflow for non-trivial tasks:
1. `architect` for minimal implementation plan
2. `coder` for the implementation
3. `analyst` when you need a repository analysis in Markdown
4. `dockerizer` when containerization is requested
5. `reviewer` for findings-first code review
6. `tester` for automated and manual verification

## Quality bar
- Smallest coherent diff
- No unrelated refactors
- Security and regression risks called out explicitly
- Clear verification commands
